The implementation

At the implementation phase, I translated the design into global variables. Those variables defined the colors, typography, type size and spacing scale that I would reuse all around.

I chose to keep things simple and just use Sass CSS with vanilla Javascript. I decided to avoid the use of any framework and use as less dependencies as possible.
