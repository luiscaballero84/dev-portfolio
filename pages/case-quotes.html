The solution

In order to be organized from the very beginning, I divided it in three phases:

  1. 1. Analysis
  2. 2. Strategy
  3. 3. Implementation

The first step was to define goals, analize the target audience, the positioning and marketing differentiation. I also narrowed my stregths, weakness and my story and mission.

After the analysis, I made a data-driven strategy, wrote and organized the content, and created the brand voice design.

Now it was time to move to the implementation phase. I created a design system, defining global variables for the colors, typography, type size and spacing scale.

For the development, I chose to keep things simple and use Sass CSS with vanilla Javascript. No frameworks and as less dependencies as possible.

Problems & Thought Process

Lessons Learned

This project helped me to solidify my React and JavaScript ES6 knowledge, specially the use of hooks and how to manage the different states.

I got great practice on how to fetch data from an API, my first hands on testing and a much better understanding of accessibility.

Before this project, my knowledge and commitment to accessibility was suboptimal. Now I have a much better grasp about accessibility issues.
